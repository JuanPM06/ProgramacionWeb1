UNIDADES DE MEDIDA EN DISEÑO WEB

Unidades Absolutas

Definición: Valores fijos que no dependen de otros elementos

Ejemplos comunes:

  • px (píxeles): Unidad básica en pantallas (ej: 16px)
  • cm (centímetros): Usado en medios impresos (ej: 5cm)
  • mm (milímetros): Precisión técnica (ej: 10mm)
  • in (pulgadas): 1in = 2.54cm (ej: 3in)
  • pt (puntos): Común en tipografía impresa (1pt ≈ 0.35mm)
  • pc (picas): 1pc = 12pt (usado en diseño editorial)

¿Cuándo usarlas?

Ideal para formatos impresos y elementos que requieren medidas exactas. El único valor usado frecuentemente en pantalla es px (píxeles). La mayoría de estos valores son más útiles cuando se usan en una salida en formato impreso que en la salida de pantalla. Por ejemplo, normalmente no usamos cm (centímetros) en pantalla. El único valor que usarás de forma frecuente es px (píxeles)

¿Como se Utilizan?

Las unidades absolutas hacen referencia a una medida física inalterable. Por ejemplo los centímetros y/o las pulgadas. Las unidades relativas dependen de otra medida. Por ejemplo pueden depender del ancho disponible en una pantalla de ordenador

Unidades Relativas

Definición: Valores que dependen de otro elemento de referencia

Ejemplos comunes:

  • %: Porcentaje del contenedor (ej: width: 50%)
  • em: Basado en el tamaño de fuente del elemento padre (ej: 1.5em)
  • rem: Relativo al tamaño de fuente de la raíz (html)
  • vw/vh: 1% del ancho (viewport width) o alto (viewport height) de la pantalla
  • vmin/vmax: Mínimo o máximo entre vw y vh

¿Cuándo usarlas?

  • Desarrollo web responsive
  • En el desarrollo de páginas web para que el texto y otros elementos escalen en relación con el tamaño de la ventana
    • En el desarrollo de productos digitales para que se adapten a diferentes dispositivos
  • En el ámbito sanitario para aproximar el consumo de recursos en un procedimiento
¿Como se Utilizan?

En el desarrollo de páginas web para que el texto y otros elementos escalen en relación con el tamaño de la ventana.
En el desarrollo de productos digitales para que se adapten a diferentes dispositivos
En el ámbito sanitario para aproximar el consumo de recursos en un procedimiento

Ejemplo de código:

.letra1 { font-size: 1.5em; }
.letra2 { font-size: 1em; }
.letra3 { font-size: 0.7em; }
